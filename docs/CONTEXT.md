Make a minecraft mod named Nametag Icons which provides icons to players nametag.

Default keybind would be N to open nametag icon menu, there will be options as followed:

1. Transparent Nametag Background : On/Off
2. Nametag Icons: On/Off
3. Choose Icon: (only available if number 2 is On) 

## Choosing number 3 will open file explorer and ask the player to choose a png file that will show in player's nametag
